# Room Spatial Configuration (房间空间配置)

本文档定义了宠物房间内各交互热点（Hotspots）的准确坐标位置，作为开发和 AI 生成代码的唯一真理来源（Source of Truth）。

## 坐标系说明

- **设计稿坐标**：通常基于左上角 (Top-Left) 为原点 (0, 0)。
- **代码实现坐标**：CSS 定位通常使用 `left` (x) 和 `bottom` (y) 百分比，其中 `bottom = 100% - top`。
- **精度**：保留小数点后两位，确保在不同分辨率下的定位准确性。

## 常见问题与纠错

### 1. 橱柜位置错误 (2026-03-06)
- **问题**：大模型曾错误地将橱柜坐标生成为接近小锅的位置 (`Left: 20%, Bottom: 25%`)。
- **原因**：缺乏明确的坐标文档，导致模型根据语义猜测或使用了错误的参考值。
- **修正**：已强制更新为用户指定的准确坐标 (`Left: 30.48%, Top: 24.07%` -> `Bottom: 75.93%`)。

### 2. 原型与正式版不一致
- `index.html` (正式版) 与 `prototype.html` (原型) 的坐标存在显著差异。
- **原则**：以 `index.html` 中的配置为准。`prototype.html` 仅作参考，不用于生产环境。

## 维护指南
- 修改任何热点位置时，请先更新本文档。
- AI 生成代码时，必须读取本文档中的坐标表，严禁通过“猜测”或“视觉估算”生成坐标。
